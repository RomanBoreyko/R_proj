# TRADING SYSTEM — INDEX

> **Два направления.** Арбитраж под прикрытием опционов. Гамма-скальп спреда ребалансом.

---

## 🎯 Два направления работы

```
D1 — АРБИТРАЖ ПОД ПРИКРЫТИЕМ ОПЦИОНОВ (Bybit / Coinbase / Deribit)
     Без опциона не работаем. Опцион = контрактный якорь + легальная база + конструктор.
     Где на актив есть опцион — там и арбитражим (паритет, календарь, фандинг, кросс-биржа).
     → [[03-STRATEGIES/D1-арбитраж-под-прикрытием-опционов]]

D2 — ETH PERP-LONG + PUT-LONG, РЕБАЛАНС (Bybit)
     Дельта-нейтральный гамма-скальп: перп=0.5× пута, ловим качели ребалансом.
     Тезис realized > implied. Тета = аренда. Пол пута = защита от взрыва.
     → [[03-STRATEGIES/D2-eth-perp-put-ребаланс]]
```

---

## Философия

Система не угадывает направление. Система **собирает премии** за:
- Схождение под контрактным якорем (паритет/базис/календарь — D1)
- Реализованную волатильность спреда через ребаланс гаммы (D2)
- Временной распад / фандинг-дифференциал (обёртки вокруг одного индекса)

Дисциплина: лестница схождения (Контракт > Структура > Статистика > Ничего) — торгуем только верх. Каждая сделка проходит [[03-STRATEGIES/spread-screening-card|карточку скрининга]].
