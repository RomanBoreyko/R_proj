## Принцип — опцион как ПРИКРЫТИЕ

Опцион на актив даёт три вещи, которых нет у голого кросс-биржевого спреда:
```
1. КОНТРАКТНЫЙ ЯКОРЬ. Опцион гаснет к индексу в фиксированный deliveryTime (кэш, европейский).
   Синт-форвард K+C−P пиннит спред к датированному схождению — не «обычно вернётся», а ПО СПЕКЕ.
2. ЛЕГАЛЬНАЯ БАЗА. Документированный кэш-сеттл к 30-мин среднему индекса (вкладка «Право ⚖️»).
   Это превращает «ставку на схождение» в «ожидание исполнения контракта».
3. КОНСТРУКТОР. Из опционов лепится синтетика (форвард, бокс, спред) — вторая/третья нога,
   которой можно заменить мёртвый/дорогой стакан фьюча или хеджить кросс-биржевой разрыв.
```
**Правило входа:** на активе НЕТ ликвидного опциона → актив вне работы. Опцион — это не нога ради греков, это **юридическо-контрактное прикрытие** всей связки.

## Карта бирж — где есть прикрытие

```
BYBIT     опционы BTC/ETH/SOL (USDT/USDC) + перпы + датированные фьючи + инверс.
          → основное поле: паритет (синт↔фьюч), календарь, cash-and-carry, фандинг.
DERIBIT   глубочайшие опционы BTC/ETH + фьючи. Эталон вол-поверхности/скоса/DVOL.
          → кросс-биржевой опцион↔опцион и опцион↔фьюч; референс IV.
COINBASE  спот + перпы (Advanced/International). Глубоких опционов НЕТ.
          → только ЦЕНОВАЯ нога (спот/перп) в кросс-биржевом спреде,
            прикрытая опционом с Bybit/Deribit. Сам по себе — не арбитражим.
```

## Какие связки (все — под опционным прикрытием)

| Связка | Где опцион-якорь | Что собираем |
|---|---|---|
| Синт↔фьюч одной серии (Bybit) | сам синт | паритет-разрыв, контрактный замок @T |
| Синт↔перп (Bybit) | синт | carry vs фандинг (фандинг-дифф) |
| Календарь синт-форвардов | оба синта | терм-структура carry |
| Bybit-цена ↔ Coinbase-цена | опцион Bybit как якорь/хедж | кросс-биржевой базис, прикрытый синтетикой |
| Bybit-опцион ↔ Deribit-опцион | оба опционы | скос/вола/паритет между площадками |

Дисциплина по [[03-STRATEGIES/spread-screening-card|карточке]]: торгуем только строки **КОНТРАКТ / СТРУКТУРА** лестницы схождения. «Статистика/Ничего» — мимо.

## Инструментарий

```
Право ⚖️ (калк)     — контрактный/легальный якорь: live deliveryTime, индекс, settleCoin.
Опцион 📈 (калк)    — двухногий коллектор: синт-спред, кросс-биржа миксом (опцион÷перп÷спот÷инверс).
Спред 🛠 (калк)     — μ/σ/Z любой пары + детектор мёртвой ноги.
cal_synth_arb.py    — скан паритета синт↔фьюч по сериям (rev/con исполнимый край).
parity_scan/log.py  — разовый скан + коллектор истории (у опционов истории нет → копим сами).
basis_desk.py       — базис спот/перп, календарь, синт по корзине + инверс-секция.
funding_scan / iv_rv— фандинг-дифф; IV vs RV.
графики-вокруг-индекса — как строить графики спредов вокруг индекса.
```

## [CON] — честно про эдж

```
Ликвидные ближние связки на Bybit/Deribit ОЦЕНЕНЫ ЭФФЕКТИВНО (ММ гоняют те же проверки).
Лок-арб = МГНОВЕННОЕ событие → вотчер по BID/ASK + RFQ-0-комиссии, иначе фи съедает.
Дальние/тонкие серии = ФАНТОМ (широкий стакан врёт mid). Фильтруй по исполнимому bid/ask.
Розничный хлеб — не лок-арб, а RV (скос/терм/паритет-дрейф) как ПОЗИЦИЯ под опционным прикрытием.
```
