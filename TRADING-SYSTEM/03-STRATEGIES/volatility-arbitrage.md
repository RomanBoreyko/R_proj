# АРБИТРАЖ ВОЛАТИЛЬНОСТИ

---

## Концепция

```
Волатильность = цена неопределённости
Implied Vol (IV) = рыночная оценка будущей vol
Realized Vol (RV) = фактическая историческая vol

IV Premium = IV - RV > 0 (исторически, в среднем 2–5 pp)
→ Опционы систематически переоценены → их выгодно продавать
```

---

## Три вида арбитража волатильности

### 1. IV vs RV (основной)

```
RV_30d = √(252/30 × Σ(ln(S_t/S_{t-1})²))  — считаем за 30 дней

Вход: IV_30d > RV_30d × 1.15  (IV завышена на 15%+)
Позиция: Продать стрэнгл (OTM call + OTM put) с дельта-хеджем
Выход: Экспирация или при IV/RV → 1.0

PnL ≈ (IV - RV) × Vega × √T
```

### 2. Межбиржевой IV спред (Deribit vs Bybit)

```
Одинаковый актив, одинаковый страйк, одинаковая экспирация
→ Должны иметь одинаковую IV

На практике: Deribit IV ≠ Bybit IV

Если Δ_IV > 2–3%:
  Продать на бирже с высокой IV
  Купить на бирже с низкой IV
  Дельта-хеджировать фьючом

Прибыль при схождении = ΔIV_вход × Vega × Лот
```

### 3. Межактивный vol спред (XAUT vs крипто)

```
Историческое соотношение IV:
  IV_XAUT / IV_ETH ~ 0.4–0.6 (золото тише крипто)

При стрессе: крипто-vol скачет, золото-vol тоже растёт но меньше
→ Ratio может отклониться от нормы

Вход: Ratio > Среднее + 2σ (золото стало относительно дорогим)
  → Продать XAUT vol, купить ETH vol (или наоборот)
Выход: Ratio → среднее

Позиция дельта-нейтральна по обоим активам
```

## Гамма-скальпинг как арбитраж vol

```
Суть: покупаем опцион (платим IV) + хеджируем дельту каждые N часов (собираем RV)

Прибыльно если: RV > IV (реальные движения превышают ожидания рынка)
Убыточно если: RV < IV (рынок стоит тихо)

P&L за период:
  = Σ(Γ × ΔS² / 2) - Θ × T
  = RV_realized × Vega × T / (2 × √T) - IV × Vega × T / (2 × √T)
  = (RV - IV) × Vega × √T / 2
```

## Мониторинг vol спредов

### Ежедневный чек

```
1. IV текущая → Deribit API: public/ticker для ATM опциона 30d
2. RV текущая → считаем по ценовому ряду (последние 30 торговых дней)
3. IV/RV ratio → если > 1.2: смотрим вход
4. Deribit IV vs Bybit IV → если ΔIV > 3%: арбитраж
5. XAUT/ETH vol ratio → если отклонение > 2σ: межактивный арбитраж
```

### В Excel (лист VOL_MONITOR — создать)

```
Колонки:
  date | asset | iv_30d | rv_30d | iv_rv_ratio | deribit_iv | bybit_iv | iv_arb_spread | action
```

## Риски арбитража волатильности

```
1. Vol может расти бесконечно → убыток шорт-vol неограничен
   Защита: купленные путы (лонг vol) как хвостовая страховка
   Размер: шорт-vol позиции ≤ 5% портфеля

2. Корреляция активов при стрессе → все активы коррелируют
   Защита: XAUT как некоррелированный актив

3. Ликвидность: Bybit options менее ликвидны → проскальзывание
   Решение: только Deribit для IV арбитража

4. Маржинальные требования на двух биржах одновременно
   Решение: резервный USDT буфер 10%
```
