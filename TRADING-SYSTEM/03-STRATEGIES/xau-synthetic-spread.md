# XAU — СИНТЕТИЧЕСКИЙ СПРЕД + ВОЛА

> Спот-лонг золота + перп-шорт → синтетический спред. Торгуем **волатильность спреда** стренглами, края покрыты. Позже — добор календарей к споту и фьючу.

Связано: [[03-STRATEGIES/volatility-arbitrage]] · [[03-STRATEGIES/spread-collection]] · [[03-STRATEGIES/time-trading]] · [[06-RISK/risk-rules]] · [[04-FORMULAS/profit-formulas]]

---

## Концепция

```
Нога A (лонг)  : спот золота      — XAUT (BYBIT:XAUTUSDT / .P)
Нога B (шорт)  : перп золота      — XAU  (BYBIT:XAUUSDT.P)

Синтетический спред = A - B  (базис спот/перп)

Спред дышит вокруг своего среднего (контанго/бэквордация + фандинг).
Направление золота захеджировано → торгуем НЕ цену, а ВОЛУ спреда.
```

Дельта по золоту ≈ 0 (лонг спот компенсирован шорт перпом). Доход — из:
- фандинга по перп-шорту (когда фандинг положительный — платят нам),
- базиса при схождении спреда к среднему,
- проданной волатильности спреда (тета стренглов).

---

## Конструкция позиции

```
1. Ядро (дельта-хедж):
   + спот XAUT            (лонг)
   - перп XAUUSDT.P       (шорт, тот же номинал)

2. Вола (на спреде/опционах золота):
   - проданный стрэнгл вокруг центра спреда   ← собираем тету
   + крылья дальше по краям (покрытие)        ← ограничиваем хвост

3. Добор (позже):
   + календари к споту и фьючу                ← раскорреляция по времени
```

## Спред и его статистика

```
spread   = A - B                  ($ или % от B)
mean     = SMA(spread, N)         N = 200 по умолчанию
sigma    = STDEV(spread, N)
z        = (spread - mean) / sigma

Центр стренгла (проданные страйки):  mean ± k_in · sigma   (k_in ≈ 1.0)
Крылья (покрытие):                    mean ± k_out · sigma  (k_out ≈ 2.0)
```

Визуализация — Pine-индикатор **«Синт. спред + вола (стренглы)»**:
`TRADING-SYSTEM\scripts\pine\synthetic_spread_vol.pine`
(вешать на график `XAUUSDT.P`, нога B = `XAUTUSDT.P`; бэнды показывают зоны проданного центра и крыльев, таблица — Z, сигма, вола).

## Правила входа

```
Продаём волу спреда, когда она ДЁШЕВА:
  • вола спреда (STDEV ретёрнов) в нижнем перцентиле своего окна
  • |z| < 0.5  (спред у среднего → симметричный стрэнгл)
  • фандинг по перпу не против нас (желательно положительный)

Страйки:
  • проданные — на mean ± 1σ
  • крылья    — на mean ± 2σ
```

## Правила выхода / управления

```
• Спред уходит к ±1σ  → роллируем проданную ногу, поджимаем крыло
• Спред пробивает ±2σ → крыло работает как стоп, фиксируем
• Вола спреда взлетела (верхний перцентиль) → не доливаем шорт-волу
• Экспирация близко → роллируем тету в следующую серию (см. [[03-STRATEGIES/time-trading]])
```

## Риски

```
1. Скачок волы спреда → проданный центр в минусе
   Защита: крылья (k_out), размер шорт-волы ограничен (см. [[06-RISK/risk-rules]])

2. Разъезд спот/перп (делистинг, депег XAUT, разные часы ликвидности)
   Защита: лимит на размер ноги, мониторинг базиса

3. Фандинг развернулся против перп-шорта
   Защита: пересчёт нетто-кэрри, при устойчивом минусе — сократить

4. Маржа на едином аккаунте под все ноги одновременно
   Защита: буфер USDT, контроль IM/MM
```
