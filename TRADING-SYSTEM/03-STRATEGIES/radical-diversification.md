# РАДИКАЛЬНАЯ ДИВЕРСИФИКАЦИЯ

---

## Принцип

```
Не предсказывать → рассеивать риск по максимально некоррелированным источникам дохода

Портфель из N некоррелированных источников → Sharpe × √N
```

### Уровень 2: Таймфреймы (3 независимых горизонта)

```
1H бот: зарабатывает на микро-волатильности
1D бот: зарабатывает на дневных колебаниях
1W бот: зарабатывает на недельных трендах

При боковике на 1D → 1H активен
При тренде на 1W → 1W сетка движется в сторону тренда
```

---

### Уровень 3: Стратегии (независимые источники PnL)

```
Grid PnL:       зависит от price movement
Option theta:   зависит от time passage (≠ price)
Basis arb:      зависит от basis convergence (≠ direction)
IV arb:         зависит от IV convergence (≠ direction)
Rebalance:      зависит от cross-asset divergence (≠ single asset)
Deposit yield:  зависит от USDT rate (≠ crypto)
```

---

### Уровень 4: Платформы (3 биржи)

```
Bybit → основной генератор (сетки, опции, депозиты)
Deribit → арбитраж волатильности, календари
Coinbase → спотовый арбитраж, ребалансировка

Риск контрагента: распределён по трём платформам
```

## Ребалансировочный протокол

### Еженедельный (мониторинг)
```
1. Открыть Excel → лист PORTFOLIO
2. Проверить текущие веса каждого актива
3. Если |Вес_текущий - Вес_целевой| > 5% → запланировать ребаланс
```

### Ребалансировочная сделка
```
Пример: ETH вырос, теперь 32% вместо 25%
  Продать ETH на 7% от портфеля
  Распределить по отставшим активам пропорционально

Инструмент: Coinbase spot (или Bybit spot)
Запись: тип = Rebalance в Excel
```

### Мониторинг корреляций
```
Раз в месяц считать 30-дневную rolling correlation:
  XAUT-ETH, XAUT-SOL, ETH-SOL, ETH-XRP

При росте корреляции выше 0.8:
  Снизить вес высококоррелированных активов
  Увеличить вес некоррелированных (XAUT, стейблы)
```

## Ожидаемые метрики диверсификации

```
Цель по Sharpe ratio: > 1.5
Цель по Max Drawdown: < 15%
Цель по APR: 20–40% годовых (от суммы спредов + тета + сетки)

Sharpe = (APR - Risk_free) / Volatility_портфеля
         = (30% - 5%) / ~17% ≈ 1.5
```
