# C6+ — ПАРИТЕТНЫЙ РЕВЕРСАЛ С ТЕТА-ОБЁРТКОЙ

> Арбитраж на контрактное схождение (паритет пут-колл), собираемый сетками по трём
> ногам вокруг оси-индекса + ручной добор в перекос. Срочник без ролирования.

Связано: [[03-STRATEGIES/basis-desk-родословная]] · [[03-STRATEGIES/spread-screening-card]] · [[06-RISK/risk-rules]] · [[scripts/portfolio/README|basis_desk / синт-фьюч]] · [[scripts/webapp/README|калькулятор: Право ⚖️]]

## Юридический якорь

Схождение обеспечено **спецификацией контракта Bybit** (не регулятором — биржа офшор):
опцион и датированный фьюч одной экспирации гаснут в один `deliveryTime` (8:00 UTC)
к одному индексу (30-мин среднее) в одной `settleCoin`. Сила = целостность сеттл-движка
Bybit (контрагентский риск), не суверенный клиринг. Цитата Bybit: «European-style
cash-settled Options… exercised only at expiration… settlement = average Index Price
within the last 30 minutes». Проверка: вкладка «Право ⚖️», кнопка «Доказать совпадение дат».

## Ось всего — ИНДЕКС

```
indexPrice = композит спот-цен по нескольким биржам (НЕ Bybit-спот).
Опцион → гаснет к индексу.  Фьюч → гаснет к индексу.  Перп → привязан фандингом.
Все три = разные обёртки вокруг ОДНОГО индекса. Базис = как каждая обёртка
сейчас оценивает один и тот же индекс. Сходятся не друг к другу — к индексу.
```
Наблюдать через график «отклонение от индекса, %»: перп прижат (±0.02%), фьюч дышит
широко (±0.3–0.5%) — **сетка работает на ФЬЮЧЕ/синтетике, не на перпе.**

## Слой 1 — ЯДРО: паритет-реверсал

```
LONG  call ATM (K, T)    Δ≈+0.5    (коллы лонговые — выбор направления синтетики)
SHORT put  ATM (K, T)    Δ≈+0.5
  = СИНТ-ЛОНГ форвард к T  Δ≈+1, Γ≈0, Θ≈0, V≈0   (премии гасятся, чистая дельта)
SHORT линейный фьюч (T)  Δ≈−1
─────────────────────────────────────────
НЕТТО: Δ≈0 · P&L = (фьюч−синт) на входе, ЗАФИКСИРОВАН, реализуется на сеттле T
```
Направление: реверсал (long синт / short фьюч) если синтетика дешевле фьюча; конверсия
наоборот. **Фьюч — линейный дат.фьюч, шортится.** НЕ inverse (coin-margined мёртв на
Bybit + 1/S не гасит линейную синтетику).

## Слой 2 — ТЕТА-ОБЁРТКА: проданные края ≤90д

`SELL OTM call + SELL OTM put` (стрэнгл ≤90д) → +Θ доход, −Γ, −V. Поверх фикс-паритета.

## Слой 3 — ТЕТА-БУСТ: короче по сроку

Θ мало → продавай края **короче** (тета ∝ 1/√T, у ближних тета/день выше). Баланс:
короткая тета жирнее, но гамма-риск острее у экспирации.

## Слой 4 — ХВОСТОВОЕ КРЫЛО: стрэнгл >90д

Цена у проданного края → BUY стрэнгл (>90д): +Γ, +V, дешёвая аренда/день. **Пред-размещать**
вместе с продажей краёв (risk-rules: проданное сразу с крылом), не «когда подойдёт».
⚠️ Дальние опционы Bybit тонки — проверь спред («Спред 🛠»); возможно крыло на 30-60д.

## Сборка СЕТКАМИ (три ноги вокруг индекса)

```
LONG-сетка  call ATM   — покупаешь колл на провале его цены
SHORT-сетка put ATM    — продаёшь пут на всплеске
SHORT-сетка фьюч        — продаёшь фьюч когда базис высоко к индексу
+ ручной добор в перекос, когда сетка не дорабатывает
```
⚠️ **Риск перекоса дельты:** три ноги гридятся независимо → в моменте Δ≠0
(+3 колла / −1 пут / −2 фьюча). Контроль: координировать шаги ИЛИ принять дрейф и
доливать руками. График-ось-индекс показывает, какую ногу догрузить.

## Параметры — конструкция в целом

| Параметр | Значение |
|---|---|
| Δ нетто | ≈0 (ядро); транзиентный дрейф от несинхронных сеток |
| Γ нетто | − (проданные края) → + (крыло) |
| Θ нетто | **+ (цель)**, но \|Θ\| < 1% эквити/день |
| V нетто | слегка − |
| Схождение | контрактное @ deliveryTime T |
| Макс. прибыль | паритет-разрыв + собранная тета + сеточный скальп − аренда крыла |
| Макс. убыток | ограничен крылом |
| Маржа | short put + проданные края; ядро под portfolio margin дёшево (Δ-офсет) |
| Кост (maker) | ядро 0.04% + фьюч 0.02% + края 0.04% ≈ 0.10% |

## Параметры — каждая нога

```
ФЬЮЧ short (T):    Δ−1 Γ0 Θ0 V0 · сходится к индексу · маржа да
CALL buy ATM:      Δ+0.5 Γ+ Θ− V+ · верх синтетики
PUT sell ATM:      Δ+0.5 Γ− Θ+ V− · низ синтетики → пара = синт-лонг Δ+1, греки 0
КРАЯ ≤90д (sell):  Δ0 Γ− Θ+(доход) V− · тета-двигатель · маржа да
КРЫЛО >90д (buy):  Δ0 Γ+ Θ−(мал) V+ · страховка · стоимость премия
```

## Синт-фьюч: параметр выбора инструмента (перп vs синтетика)

```
implied carry = (синт/индекс − 1)        фандинг-эквивалент = carry/(DTE×3)
если фандинг-эквивалент СИНТ < фандинг ПЕРП → лонг через синтетику дешевле (+фикс +датир)
Сейчас (ETH 19JUN): синт-carry 0.0028%/8ч < перп-фандинг 0.0045%/8ч → синтетика.
```

## Актив для обкатки: ETH

Глубочайшие ближние опционы + ликвидный фьюч + тугие спреды → все 3 ноги исполнимы.

## Инструменты мониторинга

```
basis_desk.py     — скринер базиса (спот/перп, календарь, паритет) по корзине
синт-фьюч калк    — implied carry, фандинг-эквивалент, греки, кост
«Право ⚖️»        — контрактный якорь + live-сверка deliveryTime + три цены вокруг индекса
«Спред 🛠»        — μ/σ/Z любой пары + детектор мёртвой ноги
parity_log.py     — пишет синт/перп/базис в CSV (история ряда)
spot_perp_basis.pine — Z-осциллятор базиса на TV + алерты ±2σ
```
