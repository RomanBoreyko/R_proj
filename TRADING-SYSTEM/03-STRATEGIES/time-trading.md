# ТОРГОВЛЯ ВРЕМЕНЕМ

> Время — единственный актив без риска реверса. Оно всегда движется вперёд.

---

## Три механизма торговли временем

### 1. Тета-распад опционов

```
Каждый день опцион теряет стоимость → продавец получает эту стоимость

Theta = dV/dt  (изменение цены опциона за единицу времени)

Максимальный тета-распад → ATM опционы, близкие к экспирации
Наша позиция: продаём опционы ВОКРУГ сеток → получаем тету ежедневно
```

**Что ускоряет тету:**
- Приближение к экспирации (тета нелинейна — ускоряется в последние 30 дней)
- Опцион ATM (около денег)
- Высокая IV (высокая vol → дороже опцион → больше тета в $)

---

### 2. Базисный распад (Futures Basis)

```
Фьюч_цена = Спот_цена × (1 + r × T/365)
где r = ставка фондирования рынка, T = дней до экспирации

Basis = Фьюч - Спот > 0 (при нормальном контанго)
Basis → 0 при T → 0 (экспирация)

Торговля: купить спот (через Coinbase) + продать фьюч (на Deribit)
          → получить базис при конвергенции
```

---

### 3. Календарный спред (Term Structure)

```
Term Structure = кривая IV по экспирациям

Нормальная: IV_ближний > IV_дальний (нисходящая кривая)
→ Ближний опцион теряет стоимость быстрее

Торговля:
  Продать ближнюю экспирацию (высокий тета)
  Купить дальнюю экспирацию (низкий тета, защита)
  Нетто: получаем разницу в тета-распаде
```

## Временная структура доходов системы

| Горизонт | Источник | Частота |
|----------|---------|---------|
| Часы | Сетка 1H (grid fills) | Каждые N часов |
| Дни | Тета проданных опций | Ежедневно |
| Недели | Сетка 1D | Несколько раз в неделю |
| Месяцы | Экспирация месячных опций | 12 раз в год |
| Кварталы | Ребалансировка 1W сеток | 4 раза в год |
| Год | Экспирация годовых опций | 1 раз в год |

---

## Формула суммарного временного дохода

```
Time_Income_Day =
    Grid_PnL_1H          (сетки 1H за день)
  + Grid_PnL_1D          (сетки 1D за день)
  + Theta_sold_calls     (тета проданных колов)
  + Theta_sold_puts      (тета проданных путов)
  - Theta_bought_puts    (тета купленных путов — РАСХОД)
  + Deposit_yield_day    (проценты на USDT в депозите)
  + Basis_daily          (базис фьюч-спот / дней_до_экспир)
```

## Ключевые риски торговли временем

```
1. Пробой диапазона → сетка остановилась, нет grid_pnl
   Защита: купленный PUT ограничивает убыток снизу

2. Скачок волатильности (vol crush в обратную) → IV вырос → проданные опции убыточны
   Защита: купленный PUT + ограниченный размер опций

3. Инверсия term structure → дальний дороже ближнего → кал.спред убыточен
   Защита: стоп по IV_спреду (выходим если спред меняет знак)
```
