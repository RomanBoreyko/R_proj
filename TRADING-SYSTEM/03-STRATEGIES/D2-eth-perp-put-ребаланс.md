## Что это

```
перп ETHUSDT лонг  +  пут ATM лонг  =  защитный пут = СИНТЕТИЧЕСКИЙ КОЛЛ
```
При 1:1 это **направленная** ставка (Δ≈+0.5). Для тезиса «нет направления, ловлю качели» нужна
**нейтраль**:
```
перп = 0.5× дельты пута  →  Δ книги ≈ 0, лонг гамма, лонг вега, ПЛАТИШЬ тету.
```
Это **гамма-скальп**, а в терминах спреда — торговля **вол-спредом**: лонг realized / шорт implied.
(Перп+пут 1:1 — НЕ спред, направление складывается. Δ-нейтраль — спред: фактор сокращается.)

---

## Тезис — не «цена ходит», а «ходит БОЛЬШЕ ожидаемого»

```
Рынок уже заложил движение в IV пута. Реальная ставка:
  realized > implied  →  сумма скальпов > тета  →  ПРОФИТ
  realized < implied  →  скальпы < тета  →  тихий бleeд премии
Тета = АРЕНДА машины. Бесплатной гаммы нет (связка Θ↔Γ).
```

---

## Протокол ребаланса (гамма-скальп)

```
1. Нейтраль = нетто-Δ книги ≈ 0 (перп = дельте пута).
2. Триггер: цена ±1% (или |Δ| > порога).
3. Вверх: Δ стрэддла/пута уехала + → ПРОДАЁШЬ перп до нуля → запер слайс ½·Γ·(Δцены)².
   Вниз:  Δ ушла − → ПОКУПАЕШЬ перп до нуля. «То по путу, то по перпу».
4. Пут ДЕРЖИШЬ (гамма-движок), ролл у экспирации.
5. Скальпь ОДНИМ линейным перпом. Не подгоняй опционный ноционал под вес.
```
Фи на бэнде 1% ≈ 1/6 слайса — фи не проблема, связывает ТЕТА (realized vs implied).

## Сайзинг и гранулярность (вывод 18.06)

Ребаланс жив, только если **дрейф дельты за бэнд ≥ минимального лота перпа**:
```
ETH мин-лот перпа ~0.01 ETH. На теле $1000, 250-ноционал/нога:
  ≤~18 DTE  → дрейф при 1% ≥ 0.01 ETH  → 1%-ребаланс РАБОТАЕТ (бери 14-21д)
  90д+      → дрейф ~0.004 ETH < лота   → 1% НЕ ребалансишь, нужно тело ~$3-5k
BTC ($64k)  слишком грубый для $1000 (дрейф 0.0004 < лота 0.001) → нужно $10k+.
SOL         гранулярность тоньше, НО 90д+ опционов нет (макс ~42д).
```
Связывает не ноционал, а **грек-бюджет**: |Θ| < 1% тела/день, Δ≈0, IM < 50-60%.
