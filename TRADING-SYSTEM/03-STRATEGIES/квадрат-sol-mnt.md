## 🚩 Критические факты (узкие места)

```
1. MNT-СТОРОНА ТОНКАЯ/МЁРТВАЯ. Перп $3М, инверс $0М (мёртв), кросс-спот $0.05М.
   Половина «квадрата» стоит на дохлых стаканах → фантом-спреды, проскальзывание, dead-leg риск.
2. D60 НЕ СУЩЕСТВУЕТ. SOL и MNT опционы максимум 39д. «d60» невозможен → бери 39д.
3. КРОСС-ГАММА ФАНТОМНАЯ. Ратио SOL/MNT не анкорено; синт-стрэнгл из двух опц ≠ воле кросса.
   Дин-хедж кросса бleeдит на тренде (мы это разбирали). Реальная гамма — только на ОДНОМ активе (SOL).
4. КОРРЕЛЯЦИЯ ЛЕГ. Все 10 ног на 2 актива → в их нарратив-движении всё коррелирует, «диверсификация» иллюзорна.
5. ОПЕРАЦИОННАЯ ПЕРЕГРУЗКА. 5 ботов + опц-кавер + кросс-арб + ребалансер = много точек отказа на 2 актива.
```

## Операционка (утро/вечер)

```
УТРО:  греки нетто (вкладка «Портфель») · дельта≈0? · |Θ|<1%/день? · застряли ли сетки (засаженный инвентарь)?
       · опц-кавер жив (тета не сожрала)? · гранулярность ребаланса ок?
ВЕЧЕР: фиксация дневного P&L по ногам · фандинг собран/уплачен · подрезка перекосов · ролл опционов у экспирации.
ПРАВКА: сетку на мёртвой ноге — снять. Перекос дельты — добор руками. Тренд ратио — стоп, не доливай вола в шорт.
```
