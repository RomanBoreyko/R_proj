# МУЛЬТИ-АКТИВ — ОПЦИОННАЯ КНИГА (C5)

> Опционные структуры за пределами золота: ETH, SOL, BTC, MNT, DOGE, XRP. Конструкт C5 портфеля. Каждый базовый актив — отдельная мини-книга с дельта-хеджем перпом.

Связано: [[03-STRATEGIES/portfolio-operating-plan]] · [[03-STRATEGIES/options-condor-zone-hedge]] · [[05-EXCEL/accounting-structure]]

---

## Принцип (общий для всех активов C5)

```
На каждый базовый актив:
  опционная структура (стрэнгл / кондор / вертикаль / диагональ)
  + дельта-хедж перпом того же актива

Источники P&L:
  тета (продажа временного распада)
  IV/RV арбитраж (продаём IV выше RV)
  ratio-арб (если структура на двух экспирациях)

Управление:
  дельта-хедж → поддерживаем |Δ$| < порог (индивидуально на актив)
  гамма → по знаку (реверсия / продолжение), см. [[options-condor-zone-hedge]]
  ролл → при приближении к экспирации, если тета ещё есть
```

## Текущие позиции (live 2026-06-10)

### ETH — основная структура

```
Экспирация 12JUN26 (короткая):
  long  ETH-12JUN26-1700-C  ×0.2   куплен колл
  long  ETH-12JUN26-1725-C  ×0.2   куплен колл
  long  ETH-12JUN26-1675-C  ×0.1   куплен колл

Экспирация 26JUN26 (средняя):
  short ETH-26JUN26-1700-P  ×0.2   продан пут
  short ETH-26JUN26-1700-C  ×0.1   продан колл
  short ETH-26JUN26-1650-C  ×0.2   продан колл

Экспирация 26MAR27 (длинная):
  short ETH-26MAR27-1700-C  ×0.1   продан колл (дальний)

Дельта-хедж:
  short ETHUSDT perp 0.02           перп-шорт под лонг-опции

Фьюч:
  long  ETHUSDT-19JUN26 0.01        элемент календаря
```

**Структура:** куплены ближние коллы + проданы дальние → диагональный дебет-спред. Куплена ближняя вола (12JUN) против проданной дальней (26JUN/MAR27). Похоже на **диагональный календарь + обратный ratio-спред**.

### SOL — стрэнгл шорт-вола

```
Экспирация 12JUN26 (ближняя):
  long  SOL-12JUN26-70-C  ×2.0     куплен колл (покрытие)

Экспирация 31JUL26 (дальняя):
  short SOL-31JUL26-60-P  ×2.0    ┐
  short SOL-31JUL26-66-C  ×1.0    ├ проданный стрэнгл
  short SOL-31JUL26-70-C  ×1.0    ┘

Дельта-хедж:
  short SOLUSDT perp 0.1
```

**Структура:** ближний куплен колл (покрытие) + дальний продан стрэнгл = **диагональный шорт-стрэнгл с покрытием**. Собираем тету дальней серии, покрываем ближней.

---

### MNT — широкий кондор

```
Экспирация 19JUN26:
  long  MNT-19JUN26-0.52-C  ×10    куплен колл центр

Экспирация 26JUN26:
  long  MNT-26JUN26-0.54-C  ×20    куплен колл выше
  long  MNT-26JUN26-0.52-P  ×10    куплен пут
  long  MNT-26JUN26-0.45-P  ×10    куплен пут ниже
  short MNT-26JUN26-0.6-C   ×30    продан колл (кэп верх)

Дельта-хедж:
  short MNTUSDT perp 10.0
```

**Структура:** длинные центральные + проданный дальний колл = **колл-спред + куплен пут** (защита вниз). Больше похоже на **ratio call-spread + long put** — лонг-гамма в центре.

---

### BTC — диагональный спред

```
Экспирация 19JUN26:
  long  BTC-19JUN26-62000-C  ×0.01   ближний куплен колл

Экспирация 25DEC26:
  short BTC-25DEC26-60000-C  ×0.01  ┐ проданы дальние
  short BTC-25DEC26-65000-P  ×0.01  ┘
```

**Структура:** ближний куплен ATM колл + дальние проданы (колл + пут) = **диагональный спред с дальним стрэнглом**. Тета от дальней серии, покрытие ближним.

---

### DOGE / XRP — хвосты

```
DOGE-26JUN26-0.08-P  long ×100    глубокий OTM пут — хвостовое страхование
XRP-26JUN26-1.3-C    long ×10     OTM колл — направленная ставка
```

Малые позиции. DOGE пут = страховка от крэша. XRP колл = асимметричная ставка на рост.

## Управление C5

### Дельта-нейтральность по каждому активу
```
|Δ$_ETH|  < 50 USDT  (при эквити 150)
|Δ$_SOL|  < 30 USDT
|Δ$_MNT|  < 20 USDT
|Δ$_BTC|  < 30 USDT
```
Перп-шорт = основной инструмент нейтрализации.

### Тета / вега баланс
- Суммарная тета C5 должна перекрывать IV-premium (продаём дороже RV).
- Вега: при росте IV — покупаем дополнительную страховку (или фиксируем шорт-вегу).

### Ролл-дисциплина
- За 7 дней до экспирации → оценить ролл или закрытие.
- Убыточные ноги: при достижении 2× премии → закрыть.

## Риски C5

| Риск | Описание | Защита |
|------|---------|--------|
| Корреляция в кризис | ETH/SOL/BTC коллапсируют вместе | ограничить суммарный ноционал C5 |
| Ликвидность MNT/DOGE | тонкие опционы, спред широкий | исполнение только лимитами |
| Маржа | C5 занимает IM на общем аккаунте с C1-C4 | мониторить IM/Equity < 50% |
| Скрытая дельта | много ног = неочевидный суммарный Δ$ | вкладка «Портфель», ежедневно |
