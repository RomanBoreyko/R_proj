# ОПЦИОННЫЙ КОНДОР + ЗОННЫЙ ХЕДЖ ПЕРПАМИ

> Помимо синт-спреда. Опционная конструкция на XAUT + дельта-хедж двумя перпами, где стиль хеджа (реверсия / продолжение) переключается по знаку нетто-гаммы.

Связано: [[03-STRATEGIES/xau-synthetic-spread]] · [[03-STRATEGIES/volatility-arbitrage]] · [[scripts/webapp/README]] (вкладка «Структура»)

---

## Две конструкции (важно не путать)

| | Reverse iron condor | Iron condor |
|---|---|---|
| Центр (ближние страйки) | **куплен** (long) | **продан** (short) |
| Крылья (дальние страйки) | **проданы** (short) | **куплены** (long) |
| Деньги при входе | дебет | кредит |
| Профит | при **движении** к крыльям | если **стоит** в центре |
| Нетто-гамма тела | **+** (длинная) | **−** (короткая) |
| Вега | + (длинная вола) | − (короткая вола) |

«Купленный центр + проданные края» = **reverse iron condor**.

---

## Карта гаммы → стиль хеджа (ядро идеи)
Стиль ребалансировки привязан к **знаку нетто-гаммы** по зоне цены:
- **+гамма** (длинный центр) → **реверсия**: гамма-скальп реверсионным гридом (купи низ / продай верх), отбивает тету.
- **−гамма** (короткий центр / у проданного края) → **продолжение**: трендовый грид доливает по движению, перекрывая риск короткой гаммы.

---

## Что показал расчёт (вкладка «Структура»)

Гамма зависит от **срока**, и профили НЕ совпадают с наивной картой «центр/края»:

- **Reverse condor (long центр):** нетто-гамма **+ во всём теле**, к крыльям → **кэп** (плоско, гамма→0). **Короткой-гамма-крыла НЕТ.** Чистая реверсия в теле; за внешними страйками опционы плоские.
- **Iron condor (short центр):** нетто-гамма **− в центре** (продолжение) и **+ у крыльев** (реверсия), с точками переключения между.
- Чем **ближе к экспирации**, тем резче гамма концентрируется у страйков (зоны проявляются); на 14+ днях профиль гладкий.

### Следствие для «продолжения на проданном крае»
В reverse condor проданный край **не даёт короткой гаммы** — он **кэпирует**. Поэтому «продолжение, перекрывающее риск края» означает не хедж −гаммы, а **перп, продолжающий тренд ЗА кэпом**, который плоские опционы уже не ловят. Континуэйшн там — это **новая направленная ставка**, а не хедж (риск края кэпом уже ограничен).

> Карта «реверсия в центре + продолжение на краях» в чистом виде — **гибрид**: реверсия = тело reverse condor (+гамма), продолжение = перп за кэпом. Это осознанный выбор, а не свойство одной стандартной структуры.

## Правила хеджа (reverse condor)
```
Центр (между внутр. страйками, +гамма):
  → реверсионный грид (гамма-скальп). Условие профита: реализованная вола > тета/IV.
За внешним крылом (кэп, опционы плоские):
  → континуэйшн-перп ловит тренд, который опционы упустили. ОСТАНОВКА — на внешнем страйке
    (внутри кэпа доливать = плодить голую дельту).
Переключение — у внутренних страйков; держать ДЕДБЕНД, иначе гриды пилят на костах.
```

## Риски
- **Тета центра.** Длинная гамма не бесплатна — если рынок стоит, структура тает. Реверсионный скальп должен набрать realized > implied.
- **Виппсо на переключении** у страйков (дедбенд обязателен).
- **Косты.** Перп-хедж — только лимиты/мейкер.
- **Кэп-дисциплина.** Континуэйшн за внешним страйком — это голая дельта, ставить стоп.
- **XAUT-опционы тонкие** — исполнение крыльев/центра с проскальзыванием.

## Инструменты
- Опционы — **XAUT** (единственные на золото на Bybit; шаг страйка ~$25).
- Перп-хедж — XAUTUSDT / XAUUSDT.
- Карта зон/payoff/гамма — вкладка **«Структура»** калькулятора.
