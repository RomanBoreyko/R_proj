## Архитектура конструктов (целевая)

```
C1  LOCKED_SPREAD      XAUTUSDT спот (long grid) + XAUTUSDT перп (short grid)
                       P&L: базис спот↔перп + декорреляция вола XAUT
                       Риск: locked spread разъедется при экстриме

C2  STATARB_PAIR       XAUUSDT перп (SELL, mean reversion)
                       + XAGUSDT перп (BUY only, trend follow, хедж по h*)
                       P&L: mean-reversion XAU + carry/тренд XAG
                       Риск: тренд XAU без разворота

C3  SYNTH_RATIO        XAUTUSDT перп (long) + XAGUSDT перп (short h*)
                       + Risk Reversal (покрывает незахеджированную дельту)
                       P&L: XAUT/XAG ratio → μ + опц.премия RR
                       Риск: ratio уходит далеко от μ

C4  OPT_CALENDAR       XAUTUSDT near/far синт-фьюч (ATM страйк)
                       P&L: временной распад + basis-to-spot
                       Риск: ранний выход, ликвид опций

C5  MULTI_ASSET_OPT    ETH / SOL / BTC / MNT / DOGE / XRP опц. структуры
                       P&L: тета, вола, ratio-арб по каждому активу
                       Риск: мульти-актив → скрытые корреляции в кризис
```

## Связи между конструктами

```
C1 ──── независим, собирает базис XAUT
C2 ──── XAG buy-only trend хеджирует XAU sell-only reversion по h*
C3 ──── shared XAGUSDT с C2; Risk Reversal закрывает незахеджированную Δ$
C4 ──── XAUT-фьючи/опционы; конкурирует за ликвидность с C3-RR
C5 ──── отдельные мини-книги по каждому активу; в кризис всё коррелирует
```

**Суммарная нетто-дельта** — считать по ВСЕМ конструктам через вкладку «Портфель» в $-дельте, не в штуках.

## 4 сетки на одном аккаунте (принцип)

Несколько сеток на одном символе неттятся. Решение — разные инструменты/типы:

| # | Сетка | Инструмент | Тип | Почему не конфликтует |
|---|-------|-----------|-----|----------------------|
| 1 | Лонг-сетка | **XAUTUSDT спот** | long-only | спот-баланс ≠ перп |
| 2 | Шорт-сетка | **XAUTUSDT перп** | short grid | перп-позиция |
| 3 | Реверсионная | **XAUUSDT перп** | mean-rev grid | другой символ |
| 4 | Трендовая | **XAGUSDT перп** | trend grid | другой актив |

- Сетки 1+2 = C1 (locked spread). Спот ≠ перп → не неттится.
- Сетки 3+4 = C2 (stat-arb пара). Серебро хеджирует золотую реверсию.
- Опционные хедж-сетки — только на XAUT-инструментах (иначе в хедж XAUT-опционов втащишь базис ~$25 синт-спреда).
- Портфельная маржа: зачёт за коррелированные ноги (золото↔серебро) → капиталоэффективно.

## C1 — Параметры сеток (XAUTUSDT.P / XAUUSDT.P)

**Инструмент:** спред `BYBIT:XAUTUSDT.P / BYBIT:XAUUSDT.P`  
**Аллокация:** 10% от тела на оба грида суммарно (по 5% каждый)  
**Исполнение:** только лимитные ордера (мейкер 0.02%), тейкер убивает эдж  
**Комиссия round-trip:** 0.08% → минимальный шаг сетки ≥ 0.10%

### Статистика спреда (live 2026-06-10, 1м)

| Параметр | Значение |
|---|---|
| μ (среднее) | ≈ +0.05% |
| Типичный дневной диапазон | ±0.15–0.30% |
| Экстремальный выброс | до ±0.50% |
| Трендовые эпизоды | 0.80–1.00% за 2–4 часа (редко) |

### Грид 1 — Реверсия (плотный, центральный)

```
Диапазон:  −0.15% … +0.25%   (μ ± ~1.5σ)
Шаг:       0.10%
Уровней:   4–5
Логика:    sell XAUT / buy XAU  когда спред → верх (+0.15%, +0.25%)
           buy XAUT / sell XAU  когда спред → низ  (−0.05%, −0.15%)
Цель:      3–8 сделок/день
P&L/шаг:   ~$0.20–0.40 чистых на $1 000 ноционала
```

### Грид 2 — Тренд (редкий, внешний)

```
Диапазон:  −0.50% … −0.15%  и  +0.25% … +0.50%
Шаг:       0.15–0.20%
Уровней:   2–3 с каждой стороны
Логика:    открываем по направлению пробоя из реверсионного диапазона
           держим до истощения тренда / возврата в реверсионную зону
Цель:      1–3 срабатывания/неделю
P&L/шаг:   ~$1.50–3.00 на $1 000 ноционала за трендовый эпизод
```

### Зонная карта

```
+0.50% ┤  ═══ тренд-грид sell ═══
+0.40% ┤
+0.25% ┤  ─── реверсия sell ──────
+0.15% ┤       ↑↑ sell here
+0.05% ┤  ···· μ ····················
−0.05% ┤       ↓↓ buy here
−0.15% ┤  ─── реверсия buy ───────
−0.30% ┤
−0.50% ┤  ═══ тренд-грид buy ════
```

### Сайзинг по размеру тела

| Тело | 5% / грид | Размер уровня (XAUT ~$4 200) | Чистых/шаг |
|---|---|---|---|
| $500 | $25 | 0.006 XAUT | ~$0.02 |
| $1 000 | $50 | 0.012 XAUT | ~$0.04 |
| $3 000 | $150 | 0.036 XAUT | ~$0.12 |
| $5 000 | $250 | 0.060 XAUT | ~$0.20 |
| $10 000 | $500 | 0.119 XAUT | ~$0.40 |

> Bybit минимум XAUTUSDT: **0.001 XAUT** → любой размер ≥ $500 проходит.

## C2 — Stat-Arb пара: механика ног

Единственный коррелированный товар на Bybit (меди/нефти/платины нет).

| Параметр | Значение |
|---|---|
| Ликвидность XAG | ~$49–58 млн/24ч, спред 0.015% |
| Корреляция ρ | **0.882** (live, скользящая) |
| RV_XAG | **50.4%** (в 2.17× волатильнее XAUT) |
| RV_XAUT | **23.2%** |
| h* = ρ·σ_XAU/σ_XAG | **≈ 0.331–0.41** (скользящий) |
| Ratio μ / σ | 60.32 / 1.675 (1h×200) |
| Текущий Z | **+2.77** 🔴 золото дорого к серебру |

**Нога 1 — XAUUSDT перп: SELL / mean reversion**
- Продаём когда XAU дорого (Z > +1), откупаем к среднему
- Двусторонняя сетка или ручные реверсионные ноги
- Источник P&L: возврат к среднему + фандинг при шорт-перп

**Нога 2 — XAGUSDT перп: BUY ONLY / trend follow**
- Покупаем при импульсе серебра (Z < −1 → серебро дёшево), держим тренд
- Только лонг — отсутствие шорт-ноги серебра защищает от squeeze
- Диапазон сетки **~2.17× шире** золотой (пропорционально воле)
- Источник P&L: тренд + carry h*-хедж

**Баланс дельты C2 (пример $1000 ноционал):**
```
XAU sell  −$1,000  →  Δ = −$1,000
XAG buy   +$331   →  Δ = +$331  (h* = 0.331)
─────────────────────────────────
Нетто C2           Δ ≈ −$669  (лонг серебра нейтрализует ~33% XAU)
```
Оставшиеся −$669 в C2 — это намеренный шорт-золото bias реверсионной стратегии.

## C3 — Ratio + Risk Reversal: механика

**Дельта-раскладка C3 ($1000 ноционал):**
```
XAUT long perp   +$1,000  →  Δ = +$1,000
XAG short perp   −$331   →  Δ = −$331   (h* = 0.331)
─────────────────────────────────────────────
Нетто без хеджа            Δ = +$669   (открытая лонг-дельта)

Risk Reversal (покрывает +$669):
  Sell OTM Call XAUT   →  Δ ≈ −X  (шортим хвост вверх)
  Buy  OTM Put  XAUT   →  Δ ≈ −Y  (покупаем хвост вниз)
  Цель: RR даёт ≈ −$669 дельты

Нетто C3 после RR       Δ ≈ $0   (дельта-нейтраль)
```

**Управление Risk Reversal (динамически):**
- Страйки фиксированы вокруг текущего ratio ± kσ
- Работа в двух стаканах: стакан XAUT-опционов (Call) + стакан XAUT-опционов (Put)
- Покупаем/продаём **дороже теоретического BSM** → каждая сделка RR в плюс по IV
- При движении цены — ребалансируем отдельно Sell-ногу и Buy-ногу
- P&L от гаммы: если реализованная вола > проданный IV → гамма-скальп в плюс

**Почему RR, а не перп-шорт:**
XAG уже занят трендовой ногой C2. Добавить XAUT перп-шорт создало бы конфликт с C1. RR закрывает дыру опционно — и собирает при этом IV-премию.

## Дисциплина и мониторинг

1. **Маржа — главный риск сейчас.** IM 74% эквити = буфер 1.70×. Минимум: >2×, цель: >3×. Приоритет — либо докапитализация, либо закрытие части C5.

3. **Косты.** Сетки — только лимиты/мейкер. Тейкер съедает тонкий эдж.
4. **Кэп-дисциплина** (reverse condor): континуэйшн за внешним страйком — это голая дельта, ставить стоп.
5. **Фьючи не гридить** (спред 0.53%) — только под календарь C4 до схождения.
6. **XAUT/XAG ratio Z=2.86** сейчас — условия для входа C2/C3 выполнены: золото дорого к серебру.
