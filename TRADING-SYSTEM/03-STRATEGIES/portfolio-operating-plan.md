# ПОРТФЕЛЬ — ОПЕРАЦИОННЫЙ ПЛАН

> Как устроена книга и как ей управлять. Один аккаунт Bybit, портфельная (кросс) маржа.
> Два направления: D1 арбитраж под опционами, D2 гамма-скальп перп+пут.

## Операционная модель — два направления

```
D1  АРБИТРАЖ ПОД ОПЦИОНАМИ   паритет/календарь/фандинг/кросс-биржа, где есть опцион.
                            Лок-арб = мгновенный → вотчер по bid/ask + RFQ-0. RV = позиция.
D2  ETH PERP+PUT РЕБАЛАНС    дельта-нейтральный гамма-скальп, перп=0.5× пута, «то пут то перп».
                            Тезис realized>implied. Пол пута = защита.
```
Любая нога входит только под опционным прикрытием (D1) или с опционным полом (D2).

## Дисциплина входа — лестница + карточка

```
Торгуем только КОНТРАКТ / СТРУКТУРА лестницы схождения.
СТАТИСТИКА — с хеджем h* + |Z|-стоп. НИЧЕГО (мем без якоря) — МИМО.
Каждая сетка/конструкт → [[03-STRATEGIES/spread-screening-card|карточка скрининга]] до постановки.
```

## Сайзинг и гранулярность (вывод 18.06)

Ребаланс/скальп жив, только если **дрейф за бэнд ≥ минимального лота**:
```
ETH ($1730), тело $1000, 250-ноционал/нога:
  D2 1%-ребаланс работает на сроке ≤~18 DTE (бери 14-21д). 90д+ → нужно тело $3-5k.
BTC ($64k): слишком грубо для $1000 (дрейф < мин-лота) → тело $10k+.
SOL: гранулярность тоньше, НО опционов только до ~42д (90д+ нет).
Связывает не ноционал, а ГРЕК-БЮДЖЕТ: |Θ| < 1% эквити/день, нетто Δ≈0, IM < 50-60%.
```
Косты: только лимит/мейкер. Шаг сетки ≥ 1.5× round-trip костов. Тейкер убивает эдж.

## Бюджет греков и маржи

```
нетто Δ (книга)   ≈ 0 в $-дельте (суммировать по вкладке «Портфель», не в штуках)
|Θ| / день        < 1% эквити
Vega              кап под событие; D2 = лонг вега (плати тету), D1 = около нуля
IM / эквити       < 50-60%, буфер MM > 2× (цель >3×)
резерв нала       15-25% (крылья + буфер ликвидации + добор В дислокацию)
```
Диверсифицируй ЭДЖ (тета/базис/фандинг/реализ-вола), не только актив — иначе всё умрёт в один режим.

## Миграция со старой книги (legacy)

Книга держала золото (XAUT/XAG) + мульти-актив мем-структуры (ETH/SOL/MNT/BTC/DOGE/XRP опционы, HUSDT/SAHARA). Под 2 направления:
```
ЗОЛОТО (XAUT/XAG)        → сворачивать (вне D1/D2). Закрывать лимитками к схождению.
МЕМ-КОИНЫ (HUSDT/SAHARA) → закрыть (нет опциона/якоря = вне D1).
ОПЦ-СТРУКТУРЫ ETH/SOL/BTC→ оставить ликвидные под D1/D2; тонкие (MNT/DOGE/XRP) свернуть.
Перпы-хеджи             → переназначить под D1 (паритет) / D2 (ребаланс-инструмент).
ПРИОРИТЕТ: IM/эквити < 50% (докап или закрытие части), потом перестройка под 2 направления.
```
