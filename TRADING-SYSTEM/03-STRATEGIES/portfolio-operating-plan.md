## 4 сетки на ОДНОМ аккаунте (портфельная маржа)

Проблема: несколько сеток на одном символе неттятся в кашу. Решение — развести по **разным инструментам/типам стаканов**, чтобы не сталкивались, при кросс-марже с зачётом за корреляцию.

| # | Сетка | Инструмент | Тип | Почему не конфликтует |
|---|---|---|---|---|
| 1 | Синт-спред лонг | **XAUTUSDT перп** | long grid | перп-позиция |
| 2 | Синт-спред sell | **XAUUSDT перп** | short grid | другой символ |
| 3 | Реверсия (центр опц., +гамма) | **XAUTUSDT СПОТ** | long-only spot grid | спот-баланс ≠ перп → не неттится с №1 |
| 4 | Тренд/продолжение | **XAGUSDT (серебро) перп** | trend grid | другой актив, отдельный стакан |

**Ключевые принципы:**
- №3 на **споте** XAUT — поэтому уживается с перп-сеткой №1 на одном аккаунте.
- Опционные хедж-сетки — на **XAUT-инструментах** (не XAUUSDT), иначе в хедж XAUT-опционов втащишь базис ~$25 синт-спреда.
- Портфельная маржа даёт **зачёт** за коррелированные ноги (золото↔серебро) → капиталоэффективно.

## Серебро как трендовая нога (XAGUSDT)
Единственный коррелированный товар на Bybit (меди/нефти/платины нет).
- Ликвидность ~$49 млн/24ч (grid-grade), спред 0.015%.
- **Корреляция с XAUT ≈ 0.833**; **вола 47% vs золото 22% → серебро в 2.1× волатильнее**.
- **Хедж-коэффициент** (мин-вариансный): `h* = ρ·σ_gold/σ_silver = 0.833·22.2/47.3 ≈ 0.39` → **на $1 золота ≈ $0.39 серебра**.
- Диапазон/шаг серебряной сетки делать **~2× шире** золотой.
- Риск: ~17% идиосинкразии (gold/silver ratio гуляет) → несовершенный хедж.

## Снимок книги (на момент разбора, 2026-06-08)
- Эквити ~$152, **запас по марже 1.83×**, свободно ~$12 — тонко.
- Нетто **Δ$ ≈ −$1120** (направленный шорт ~7× эквити — НЕ нейтрал!), **Θ ≈ −1.7%/день** (платим распад), **Vega шорт**.
- Книга = 6 перпов/фьючей + 20 опционов (XAUT/ETH/SOL/MNT/BTC).
- **Золотого спота (XAUT) нет** — нога «спот-лонг» из синт-спреда не набрана; оба золотых перпа в шорт.
- Сценарий-убийца: рост рынка + всплеск IV (шорт-дельта + шорт-вега + шорт-гамма одновременно).

## Дисциплина и мониторинг
1. **Маржа — главный риск.** 4 сетки + опционы на буфере 1.83× = тонко. **Старт микрообъёмом.**
2. **Нетто-дельта золота** (XAU+XAUT+серебро) — контроль вкладкой «Портфель» (суммирует перпы+опционы+спот). 4 коррелированных книги легко дают скрытый перекос.
3. **Косты.** Сетки и хедж — только лимиты/мейкер. Тейкер съедает тонкий эдж.
4. **Кэп-дисциплина** опционов: континуэйшн за внешним страйком — это голая дельта, ставить стоп.
5. **Фьючи не гридить** (спред 0.53%) — только под календарь до схождения.

## Чем смотреть (вкладка «Портфель» калькулятора)
- **Мета-спред** — вся книга одной панелью: Δ$, Γ, Θ ($/день и % эквити), Vega + сценарий.
- **Спот / кошелёк** — $-оценка, флаг золотого спота.
- **Спреды / структуры** — распознавание конструкций + греки по группе.
- **Состояние спредов** — uPnL по структурам (бар).
- **Спреды портфеля — графики** — авто-график спреда по каждой паре с линией «собрано».
- **График баланса USDT** — кривая из transaction-log.
