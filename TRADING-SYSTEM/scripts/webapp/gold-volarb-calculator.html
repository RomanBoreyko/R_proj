C1 — Locked Spread: XAUTUSDT спот (long) + XAUTUSDT перп (short)
Конструкт: лонг спот + шорт перп одного актива = locked spread. Доход: базис спот↔перп + декорреляция волатильности XAUT.
P&L: при схождении basis → 0 фиксируем. Фандинг перп-шорта добавляет (или вычитает) carry.
Риск: locked spread разъедется при делистинге или экстремальном событии. Маржа занята полным ноционалом спота.
TradingView: стандартный виджет показывает live-свечи. Для своего сохранённого графика — вставь URL (Share → Published chart URL).

C2 — Стат-арб XAU реверсия + XAG трендовый хедж
Реверсионная нога: XAUUSDT перп, входим по Z-score ≥ k (среднее возврата по AR(1), полужизнь ~38 баров).
Трендовая нога (хедж): XAGUSDT перп. При тренде золота уходит в просадку реверсия → серебряная нога компенсирует.
h* = ρ × σ_XAU / σ_XAG ≈ 0.41 — min-variance hedge ratio. На $1 ноционала XAU → $0.41 XAG.
XAUT/XAG ratio Z = +2.86 (на 2026-06-10) — золото дорого к серебру, входной сигнал C2/C3.

C3 — Synth Ratio XAUT/XAG + опционная обёртка
Ноги: лонг XAUTUSDT перп + шорт XAGUSDT перп (по h*) = ratio convergence к μ.
Опционная обёртка: купленные OTM XAUT-опционы (защита) + проданные OTM (cover, тета).
Синт-фьюч (C4-связь): long call(K) + short put(K) ATM = эквивалент фьючу, страйк ≈ near forward.
Вол-арб: продаём волу когда IV/RV > 1.2 (20% premium к реализованной).

C4 — Опц. календарь XAUTUSDT (ATM синт-фьюч)
Синт-фьюч: long call(K) + short put(K), K ≈ ATM forward. Near/far экспирация → calendar spread.
P&L: near converges to spot at expiry → basis collapses → собираем carry. Плюс при тихом рынке (тета near > тета far).
Страйк: ATM forward (≈ текущий near-фьюч). OTM добавляет skew-экспозицию, усложняет учёт.
XAUTUSDT фьючи на Bybit: 12JUN26, 19JUN26, 26JUN26, 31JUL26. Спред 0.53% — не гридить, только calendar до схождения.

Опц. синт-фьюч (ATM, вручную)

Структура — payoff + карта гаммы + зоны хеджа
Reverse condor (куплен центр): +γ на всём теле → реверсионный грид. За крыльями — кэп, опционы плоские.
Iron condor (продан центр): −γ в теле (трендовый грид) → +γ у крыльев (реверсия). Точки переключения = нули гаммы.
Дедбенд у точек переключения обязателен — иначе гриды пилят на костах.
