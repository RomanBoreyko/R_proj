# КОНЦЕПЦИЯ СИСТЕМЫ

## Три кита

### 1. Торговля временем
Время — единственный актив, который всегда движется в одну сторону.

- **Опционный тета-распад**: проданные опционы теряют стоимость каждый день
- **Календарный спред**: дальний фьюч дороже ближнего → сходятся к экспирации
- **Базисный спред (спот–фьюч)**: фьюч торгуется с премией → конвергенция к экспирации

```
Прибыль_времени = Тета_продан × дни + Базис_фьюча / дней_до_экспир
```

---

### 2. Арбитраж волатильности

Волатильность одного инструмента расходится с другим → собираем спред при возврате.

- **IV спред**: дерибит-опционы vs байбит-опционы на один актив
- **Календарный спред IV**: ближний IV vs дальний IV (term structure)
- **Межактивная волатильность**: XAUT vs ETH vs BTC vol ratio

```
Vol_арб_PnL = (IV_продал - IV_купил) × Vega × Лот
```

---

### 3. Радикальная диверсификация + Ребалансировка

Не концентрация — рассеивание. Не удержание — ребалансировка.

- 5 активов × 3 таймфрейма сеток = 15 независимых источников прибыли
- Опционная обёртка каждого бота = ещё один слой дохода
- Ребалансировка при отклонении → автоматическая покупка дешёвого, продажа дорогого

```
Вес_актива = Целевой_вес ± Полоса_допуска (±5%)
Ребаланс_триггер = |Текущий_вес - Целевой_вес| > 5%
```

## Карта прибыли

| Источник | Инструмент | Частота | Платформа |
|----------|-----------|---------|-----------|
| Сетка (буквальная прибыль) | Фьюч-сетка | Внутри дня | Bybit |
| Тета-распад проданных опционов | Проданный кол + пут | Ежедневно | Bybit |
| Защита купленным путом | Купленный пут | При падении | Bybit |
| Депозитный доход | Spoт + депозит | Ежедневно | Bybit |
| Базис фьюч-спот | Спот vs фьюч | К экспирации | Bybit + Deribit |
| Календарный спред | Дальний vs ближний | К экспирации | Deribit |
| Арбитраж IV | Опционы Deribit vs Bybit | Эпизодически | Deribit + Bybit |
| Ребалансировочная прибыль | Спот-ребаланс | При отклонении | Coinbase + Bybit |
| Гамма-скальпинг | Дельта-хедж опций | Внутри дня | Deribit |

---

## Правила системы (абсолютные)

```
❌ Шортов нет (ни фьюч, ни спот)
❌ Плечо > 2× — запрещено
❌ Уровни ликвидации — не допускаются
❌ Фандинг не трогать как стратегию
✅ Каждая операция → запись в Excel
✅ Опционная обёртка = защита + доход
✅ Ребалансировка при отклонении >5%
```
