> Не угадываем направление. Собираем **схождение под контрактным якорем** и **реализованную
> волатильность спреда через ребаланс**. Два направления, один принцип.

## Единый принцип

Любая сделка системы = **спред** (две претензии на один/коррелированный фактор; рыночное направление сокращается, торгуем остаток). Прибыль — из остатка:
```
СХОЖДЕНИЕ под якорем   → паритет/базис/календарь/фандинг (остаток = базис)
ВОЛАТИЛЬНОСТЬ спреда   → гамма-скальп ребалансом (остаток = realized − implied)
```
Дельта-нейтрально. Не «куда пойдёт цена», а «как сходятся обёртки одного индекса».

## Два направления

```
D1 — АРБИТРАЖ ПОД ПРИКРЫТИЕМ ОПЦИОНОВ (Bybit/Coinbase/Deribit)
     Без опциона не работаем. Опцион = контрактный якорь схождения + легальная база + конструктор.
     Собираем: паритет (синт↔фьюч), календарь, cash-and-carry, фандинг-дифф, кросс-биржевой базис.
     → [[03-STRATEGIES/D1-арбитраж-под-прикрытием-опционов]]

D2 — ETH PERP-LONG + PUT-LONG, РЕБАЛАНС (Bybit)
     Дельта-нейтральный гамма-скальп (перп=0.5× пута). Ловим качели ребалансом «то пут, то перп».
     Тезис realized > implied. Тета = аренда машины. Пол пута = защита от взрыва.
     → [[03-STRATEGIES/D2-eth-perp-put-ребаланс]]
```

## Лестница схождения (дисциплина входа)

```
КОНТРАКТ    паритет/сеттл/экспирация — схождение ГАРАНТИРОВАНО спекой      → торгуем
СТРУКТУРА   один актив, разные обёртки (перп/спот, лин/инв)               → торгуем + стоп
СТАТИСТИКА  корреляция активов (ETH/BTC)                                  → только с хеджем + |Z|-стоп
НИЧЕГО      мем-пары без якоря                                            → МИМО (полка, не дыхание)
```
Каждая сделка проходит [[03-STRATEGIES/spread-screening-card|карточку скрининга]]. Пустая графа = «не знаю» = стоп.

## Правила системы (абсолютные)

```
✅ Только дельта-нейтральные конструкты (нетто Δ ≈ 0 на книгу)
✅ Опцион даёт ПОЛ (ограниченный убыток) или ЯКОРЬ (контрактное схождение) — без него не лезем
✅ |Θ| < 1% эквити/день · IM < 50-60% · никаких уровней ликвидации
✅ Каждая операция → запись (Excel/лог). Каждая сетка → карточка скрининга
✅ Шорт-ноги ОК, если они хедж (фьюч-шорт в паритете, инв-шорт в фандинг-паре) — не голое направление
❌ Голое направление / голый шорт волы без крыла (урок VELVET)
❌ Доливка в минус на проданной воле в тренд (мартингейл — урок HUSDT)
❌ Плечо ради плеча · ставка на актив без опциона (D1) · реализ<implied как «обязательно вернётся» (D2)
```

---

## Активы и площадки

```
Опционы есть (работаем): BTC, ETH (Bybit+Deribit), SOL (Bybit, до ~42д).
Coinbase: ценовая нога (спот/перп) под опционным прикрытием Bybit/Deribit.
Нет ликвидного опциона → актив вне работы (D1). D2 — на ETH (90д+ опционы + ликвидный инверс).
```
