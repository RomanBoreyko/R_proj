## Режим: LONG

> Только лонги. Шортов нет. Бот открывает лонг при создании, закрывает при росте.

**Логика:**
1. Бот делит диапазон `[Нижняя, Верхняя]` на N ячеек
2. На каждой ячейке — лонг-ордер на покупку (ниже) и продажу (выше)
3. Прибыль = разница между ценой продажи и покупки за вычетом комиссии

## Параметры бота (все 5 активов × 3 таймфрейма)

### Структура параметров

| Параметр | Описание | Пример |
|----------|---------|--------|
| `pair` | Торговая пара | XAUTUSDT |
| `direction` | Режим | Long |
| `lower_price` | Нижняя граница сетки | По таймфрейму |
| `upper_price` | Верхняя граница сетки | По таймфрейму |
| `grid_count` | Количество ячеек | 20–50 |
| `grid_type` | Тип | Arithmetic / Geometric |
| `leverage` | Плечо | 1–2× |
| `investment` | Объём в USDT | — |
| `stop_loss` | Стоп (ниже нижней) | Опционально |
| `take_profit` | ТП (выше верхней) | Опционально |

### Как устанавливать диапазоны

```
1H бот → диапазон = High/Low последних 20 часовых свечей
1D бот → диапазон = High/Low последних 20 дневных свечей
1W бот → диапазон = High/Low последних 20 недельных свечей
```

## Формулы расчёта прибыли сетки

### Прибыль за одно срабатывание (одна ячейка)

```
Arithmetic:
  Шаг = (Upper - Lower) / N
  Прибыль_ячейки = Шаг - (2 × Комиссия × Цена_входа)

Geometric:
  Шаг_ratio = (Upper / Lower)^(1/N)
  Прибыль_ячейки% = (Шаг_ratio - 1) - 2 × Taker_fee
```

### Годовая прибыль сетки (упрощённая)

```
APR_сетки = (Прибыль_ячейки / Инвестиция) × Срабатываний_в_сутки × 365
```

### Срабатываний в сутки (оценка)

```
Срабатываний = (ATR_таймфрейма × Коэф / Шаг_сетки)
где:
  ATR = Average True Range за период
  Коэф = 0.5–0.8 (эффективность заполнения)
```

## Geometric vs Arithmetic

| | Arithmetic | Geometric |
|--|-----------|----------|
| Шаги | Равные в $$ | Равные в % |
| Лучше для | Боковик в узком диапазоне | Широкий волатильный диапазон |
| XAUT | ✅ Arithmetic | — |
| SOL, MNT | — | ✅ Geometric |

## API V5 — ключевые эндпоинты

```
Рынок:
  GET /v5/market/kline           → свечи для расчёта ATR
  GET /v5/market/tickers         → текущая цена

Боты (через UI, не API напрямую):
  Создание через Bybit Web/App → Trade → Trading Bot → Futures Grid

Позиции:
  GET /v5/position/list          → текущие позиции
  GET /v5/order/realtime         → активные ордера

Аккаунт:
  GET /v5/account/wallet-balance → баланс
  GET /v5/account/fee-rate       → комиссии
```

Базовый URL: `https://api.bybit.com`
Документация API V5: `https://bybit-exchange.github.io/docs/v5/intro`
